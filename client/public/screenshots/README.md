# PWA Screenshots

This directory should contain screenshots for the PWA manifest:

## Required Screenshots:

1. `home.png` (1280x720) - Home screen showing prayer countdown
2. `focus.png` (390x844) - Focus session timer

## How to create screenshots:

1. Run the app in development mode
2. Take screenshots of the home page and focus page
3. Resize them to the specified dimensions
4. Save them in this directory

## Alternative:

You can use browser dev tools to capture screenshots or use a screenshot tool that can capture specific viewport sizes.
